# Smart Monitoring Dashboard

A professional AI-powered smart monitoring dashboard web application.

## Overview
This project is a real-time monitoring system that visualizes sensor data such as motion detection, temperature changes, and activity patterns using modern UI and AI insights.

## Features

- Real-time dashboard updates
- Motion detection tracking
- Temperature analytics (high/low/average)
- Event timeline history
- Interactive charts (line + bar)
- AI Insight Panel with smart analysis
- Daily & weekly summaries
- Anomaly detection alerts
- Responsive design (mobile + desktop)
- Dark / Light mode toggle

## AI Insights

The system automatically generates insights such as:

- Peak activity hours
- Unusual motion detection
- Temperature fluctuations
- Daily activity summary
- Weekly behavioral trends
- Risk level detection (Low / Medium / High)

##  Tech Stack

- React
- Vite
- Tailwind CSS
- Firebase Realtime Database
- Chart.js

##  UI Style

- Modern SaaS dashboard design
- Minimal and clean layout
- Blue / white / dark grey theme

- Smooth animations
